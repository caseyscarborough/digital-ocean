# Digital Ocean API

This gem is a simple and easy-to-use wrapper for Digital Ocean's API.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'digital-ocean'
```

And then execute:

```bash
$ bundle install
```

Or install it yourself as:

```bash
$ gem install digital-ocean
```
